# todolist
Ola, sou Joao Francisco, aluno da materia de Projeto de Interface WEB.
Este e meu projeto no qual e feito uma pagina web onde temos uma lista de tarefas,
o publico alvo e pessoas que buscam algo simples mas que tenha uma funçao muito
importante para ajudar no dia a dia de quem esta sempre esquecendo algumas
tarefas importantes do seu dia-a-dia.

Foram implementadas telas de login, cadastro, recuperar senha, tarefas, 
adicionar/editar tarefa, detalhes da tarefa e estatisticas
